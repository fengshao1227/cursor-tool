# 独占卡密功能使用说明

## 🎯 功能说明

独占卡密功能让每个卡密都绑定一个独立的 Cursor Token，实现：
- ✅ **一卡密一账号**：每个卡密对应独立的 Cursor 账号
- ✅ **自动注入登录**：输入卡密后自动注入 Cursor 账号，无需手动登录
- ✅ **简单快捷**：只需输入卡密，软件自动完成所有配置

## 📋 使用流程

### 1. 首次使用

软件启动时会自动检查是否有卡密，如果没有会弹出卡密输入框：

```
🔐 请输入卡密激活软件
┌────────────────────────────────┐
│ 服务器地址（可选）              │
│ ▸ http://117.72.163.3:8080     │
│                                 │
│ 请输入新卡密                   │
│ ▸ XXXXX-XXXXX-XXXXX-XXXXX     │
│                                 │
│ 🚀 激活后将自动注入Cursor账号   │
│                                 │
│ [ 激活并注入账号 ]              │
└────────────────────────────────┘
```

**操作步骤：**
1. 输入你的卡密
2. 点击"激活并注入账号"
3. 等待激活成功
4. 启动 Cursor，会自动登录到对应账号

### 2. 更换卡密

在侧边栏底部，点击黄色的 **🔑 更换卡密** 按钮：

```
[ 账号管理 ]
[ 系统工具 ]
[ 备份管理 ]
[ 设置     ]
───────────
[ 🔑 更换卡密 ]  ← 点击这里
───────────
```

**操作步骤：**
1. 点击"更换卡密"按钮
2. 如果 Cursor 正在运行，会提示关闭
3. 输入新的卡密
4. 点击"激活并注入账号"
5. 启动 Cursor，自动登录到新账号

### 3. 解绑设备

如果需要更换设备，可以在更换卡密界面点击"解绑此设备"：

```
┌────────────────────────────────┐
│ ✅ 已激活                       │
│ 到期：2025-12-31               │
│                                 │
│ [ 激活并注入账号 ] [ 取消 ]     │
│ ───────────────────────────     │
│ [ 解绑此设备 ]   ← 点击解绑     │
└────────────────────────────────┘
```

## 🔧 管理员端操作

### 生成独占卡密

在管理后台 → 卡密管理：

1. 设置数量、有效期、设备数
2. ✅ **勾选"使用独占Token"**
3. 点击"生成卡密"

**注意事项：**
- 需要先添加独占 Token
- 生成 N 个卡密需要 N 个独占 Token
- 独占 Token 一旦使用就被标记为"已消耗"

### 添加独占Token

在管理后台 → Token 管理：

1. 粘贴 Cursor Token
2. ✅ **勾选"独占模式"**
3. 点击"添加Token"

## ❓ 常见问题

### Q: 为什么需要关闭 Cursor？
**A:** 注入账号信息需要修改 Cursor 的数据库文件，必须在 Cursor 关闭时才能安全操作。

### Q: 更换卡密后之前的账号还能用吗？
**A:** 更换卡密后，软件会自动切换到新卡密对应的账号，之前的卡密不会自动失效，但软件只使用最新的卡密。

### Q: 可以在多台设备上使用同一个卡密吗？
**A:** 取决于卡密的设备数限制。如果卡密支持多设备，可以在多台设备上激活。

### Q: 卡密过期了怎么办？
**A:** 卡密过期后软件会提示需要重新输入卡密，使用新卡密即可。

### Q: 忘记了当前使用的是哪个卡密？
**A:** 点击"更换卡密"按钮，界面会显示当前卡密的状态和到期时间。

## 🎨 界面说明

### 侧边栏按钮
- **🔑 更换卡密**：黄色按钮，点击更换当前使用的卡密

### 状态显示
- ✅ **已激活**：卡密有效且已激活
- ⚠️ **未激活**：需要输入卡密
- 🔴 **已过期**：卡密已过期，需要更换

### 弹窗模式
- **必填模式**：首次使用时，必须输入卡密才能使用软件
- **更换模式**：主动更换卡密，可以取消操作

## 💡 使用建议

1. **定期检查卡密状态**：避免卡密突然过期影响使用
2. **保存好卡密**：建议将卡密保存在安全的地方
3. **及时更新**：收到新卡密后及时更换
4. **单设备使用**：如果卡密是单设备的，不要在多台设备上使用

## 🔒 安全提示

- 卡密是你使用软件的凭证，请妥善保管
- 不要将卡密分享给他人
- 如果卡密泄露，请及时联系管理员更换
- 软件会自动保存激活信息，重启后无需重新输入卡密
